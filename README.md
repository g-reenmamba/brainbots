# brainbots
## HostelSync: Unified Hostel Operations Platform ##  Overview HostelSync is a high-fidelity digital ecosystem designed to replace the fragmented, manual processes of hostel management (registers, WhatsApp groups) with a streamlined, real-time administrative command center. Built for the **State-Level Hackathon**, it focuses on operational efficiency, security, and student satisfaction. ##  Key Features - **Warden Command Center:** Real-time attendance tracking, maintenance monitoring, and visitor logs. - **Digital Gate Pass Workflow:** Seamless student exit requests with one-tap warden approval and automated audit trails. - **Real-time Room Allocation:** Visual floor maps and occupancy tracking for Block A/B management. - **Mess Menu & Voting:** Interactive weekly schedules with student satisfaction polls to improve culinary experiences. - **Parent Notifications:** Automated alerts for gate pass approvals (SMS-ready structure). - **Complaint Escalation:** Integrated SLA tracking for maintenance issues like plumbing and connectivity. ##  Tech Stack - **Frontend:** HTML5, CSS3 (Tailwind CSS), JavaScript (Modern ES6+) - **Design System:** HostelSync Velocity (Custom Design Language) - **Icons & Typography:** Google Material Symbols, Manrope/Work Sans Fonts - **Architecture:** Component-based UI with a focus on responsiveness and accessibility. ##  Project Preview The project currently includes the following interactive screens:
1. **Warden Dashboard:** The central hub for attendance and maintenance.
 2. **Gate Pass Approvals:** Managing and reviewing student exit requests.
 3. **Room Allocation:** Floor-wise occupancy tracking.
   4. **Mess Menu & Voting:** Weekly culinary schedules and feedback loops. ##  Getting Started To view the platform locally: 1. Clone this repository. 2. Open `index.html` (or individual screen files) in any modern web browser. 3. (Optional) Use a "Live Server" extension in VS Code for a better development experience. ##  Hackathon Context This platform was developed to address the "operational chaos" found in university housing, providing a scalable solution for state-level institutional deployment.
